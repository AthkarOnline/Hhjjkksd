دليل تنصيب موقع أذكار المسلم

خطوة بخطوة لرفع موقعك على GitHub والحصول على أفضل نتائج في محركات البحث

1

نسخ كود الموقع

انسخ الكود الكامل لموقع أذكار المسلم الذي تم إرساله إليك سابقاً.

<!DOCTYPE html>
<html lang="ar" dir="rtl">
<head>
  <meta charset="UTF-8">
  <!-- كود كامل الموقع -->
</head>
<body>
  <!-- محتوى الموقع -->
</body>
</html>
نسخ الكود الكامل
2

الدخول إلى GitHub

سجّل الدخول إلى حسابك على GitHub واذهب إلى مستودع موقعك.

تسجيل الدخول إلى GitHub فتح المستودع
3

تعديل ملف index.html

افتح ملف index.html في المستودع وانقر على أيقونة القلم للتعديل.

تعديل الملف
4

استبدال المحتوى

احذف المحتوى الحالي للملف والصق الكود الجديد الذي نسخته.

1. حدد كل المحتوى (Ctrl+A)
2. احذفه (Delete)
3. الصق الكود الجديد (Ctrl+V)
4. احفظ التغييرات
5

حفظ التغييرات

أدخل وصفاً للتحديث واحفظ التغييرات على GitHub.

رسالة الـ Commit: تحديث الموقع بالتصميم الجديد
الفرع: main
انقر على "Commit changes"
سيتم تحديث الموقع خلال 1-5 دقائق
6

زيارة الموقع

انتقل إلى موقعك الجديد وتأكد من عمله بشكل صحيح.

فتح موقعي الآن
تهانينا! موقعك الآن يعمل بنجاح

الخطوات التالية بعد التنصيب

إرسال خريطة الموقع إلى Google Search Console
مشاركة الموقع على وسائل التواصل الاجتماعي
اختبار الموقع على الهاتف والحاسوب
